## Drawing

We can simply draw it in the update method of stage component:

::: Code pathTemplate="{0}tutorial/src/engine/tutorialData/components/stage.{2}"

```ts
export const stage = {
    update() {
        const layout = new Rect({
            l: judgeLine.l,
            r: judgeLine.r,
            t: 1 - note.radius / 4,
            b: 1 + note.radius / 4,
        })

        skin.sprites.judgeLine.draw(layout, 0, 1)
    },
}
```
